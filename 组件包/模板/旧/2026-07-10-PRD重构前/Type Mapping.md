# 动态事件流 (`Event_Type`) 
发生了什么？要后续做什么？

| 标签 (`Event_Type`) | 适用场景          | 核心逻辑                      |
| ----------------- | ------------- | ------------------------- |
| `#type/meeting`   | 会议纪要、技术交底     | 讨论过程 -> 决策 -> 待办          |
| `#type/issue`     | 不具合(NCR)、现场问题 | 现场发现 -> 根因 -> 修复方案        |
| `#type/change`    | 设计变更、工程量增减    | 变更缘由 -> 影响分析(造价/工期) -> 审批 |
| `#type/risk`      | 风险预警、潜在隐患     | 识别风险 -> 等级评估 -> 应对策略      |
| `#type/review`    | 项目复盘、阶段总结     | 原定目标 -> 实际偏差 -> 经验复盘      |

# 静态参考库 (`Info_Type`) 
这是什么？有什么技术指标？

| 标签 (`Info_Type`)   | 适用场景           | 核心逻辑                 |
| ------------------ | -------------- | -------------------- |
| `#info/spec`       | 技术参数、做法标准、设备书  | 核心指标 -> 适用场景 -> 合规要求 |
| `#info/entity`     | 业主画像、供应商评价     | 能力评估 -> 合作策略 -> 关键联系 |
| `#info/compliance` | 招标文件、报批文件、合同   | 条款范围 -> 责任边界 -> 风险提示 |
| `#info/constraint` | 地质报告、环境现状、场地限制 | 物理现状 -> 防御措施 -> 施工限制 |
| `#info/asset`      | 厂房区域、机房、设备清单   | 位置属性 -> 接口要求 -> 运行维护 |
